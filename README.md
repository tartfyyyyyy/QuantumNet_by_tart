# QuantumNet_by_tart
Max‑performance Magisk module for Android internet speed and latency.


**Max‑performance Magisk module for Android internet speed and latency.**

Flash it, reboot, and your device’s network stack runs at its absolute limit.

---

## What it does
- Boosts download/upload speed
- Lowers ping and jitter
- Forces the fastest DNS
- Unlocks full Wi‑Fi bandwidth
- Removes Android’s default network bottlenecks

---

## Installation
1. Download the latest `QuantumNet_by_tart.zip` from [Releases] (https://www.up-4ever.net/dficraz8ifex)
2. Flash in Magisk Manager → Modules → Install from storage
3. Reboot

---

## Uninstall
Remove the module in Magisk and reboot – everything reverts.
